# Network Architecture

## Initial Setup

Tablet → Guest WiFi → Captive Portal → Internet

---

## Key Characteristics of Network
- Open WiFi network with captive portal authentication
- Designed for temporary guest sessions
- Not optimized for persistent devices

---

## Constraints
- No direct control over network infrastructure
- Dependent on external vendor (Blueprint)
- Corporate restrictions from Marriott

---

## Additional Configuration
- MAC address whitelisting applied by ISP
- Attempted Ethernet connection via adapter
