# Tablet Kiosk Deployment in Managed Hotel Network

## Overview
Deployed and stabilized a Samsung Galaxy Tab A8 (SM-X200) as a kiosk-based time tracking system (Paycom) in a hotel environment with a captive portal network.

This project demonstrates real-world troubleshooting in a managed network with restrictions, including session persistence issues, MAC address whitelisting, and vendor coordination.

---

## Key Skills Demonstrated
- Network troubleshooting (DHCP, captive portal, MAC whitelisting)
- Root cause analysis (application vs network issues)
- Kiosk configuration (Fully Kiosk Browser)
- Android system behavior debugging
- Vendor coordination (Blueprint ISP, Marriott Service Desk)

---

## Technologies Used
- Samsung Galaxy Tab A8 (SM-X200)
- Fully Kiosk Browser
- Android OS
- Captive Portal Network
- MAC Address Whitelisting
- Basic Networking (IP, DNS, VLAN concepts)

---

## Outcome
- Identified root cause of session instability (network-level restriction vs app issue)
- Reduced manual re-authentication frequency
- Proposed scalable solution using private VLAN / Ethernet
