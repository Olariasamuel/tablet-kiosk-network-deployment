# Lessons Learned

## Technical Insights
- Captive portals are unsuitable for kiosk devices
- Network behavior can override application settings
- Session persistence depends on network architecture

---

## Troubleshooting Approach
- Always isolate variables:
  - Device
  - Application
  - Network

- If issue persists across all → likely infrastructure-related

---

## Real-World Skills Gained
- Vendor communication in IT environments
- Understanding managed network limitations
- Translating technical issues into actionable solutions

---

## Key Takeaway
Not all problems are solvable at the device level — understanding the network layer is critical.
