# Project Overview

## Context
The hotel transitioned from a dedicated physical time clock machine to a tablet-based solution using Paycom.

The device needed to remain logged in continuously to allow employees to clock in/out without interruptions.

---

## Environment
- Hotel: SpringHill Suites
- Network: Guest WiFi (captive portal)
- Managed by: Blueprint (ISP) and Marriott corporate network policies
- Device: Samsung Galaxy Tab A8 (SM-X200)

---

## Objective
Deploy a stable kiosk system capable of:
- Maintaining persistent login sessions
- Operating without user interaction
- Functioning within a restricted network environment
